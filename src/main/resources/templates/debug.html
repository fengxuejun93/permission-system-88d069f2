6. 视角切换验证

完整复查步骤：用顶部下拉切换到不同角色，访问同一条动态，观察操作差异。
切换身份 → 访问同一条动态 → 申请/处理好友 → 修改可见范围 → 评论回复 → 查看统计变化
1) 切到张三(本人) → 看到自己动态有"隐藏/恢复/删除"操作，评论有"删除"按钮
2) 切到李四(好友) → 看到张三的"好友可见"动态，可以评论回复，但不能编辑张三的动态
3) 切到王五(非好友) → 看不到张三的"好友可见"动态，详情页显示"申请加好友"按钮
4) 切到陈管理(系统管理员) → 看到被隐藏动态的标记和"恢复"按钮，但看不到"仅自己可见"的私密动态
5) 切到周班长(班级管理员) → 同班同学的被隐藏动态可见标记，同样不能绕过私密动态
6) 好友申请：王五申请加张三 → 张三同意 → 动态列表和统计数字同步变化
7) 修改可见范围：张三把公开动态改为"仅好友" → 非好友视角列表立刻减少
8) 隐藏/恢复动态：张三隐藏一条动态 → 管理员看到标记 → 管理员恢复 → 正常列表同步
